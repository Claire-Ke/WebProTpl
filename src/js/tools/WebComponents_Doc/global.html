Effectiv4AddShadow(extends4ClassName, extends4ElemName) → {boolean}

验证是否可以有效的使用Element.attachShadow()。
PS: 满足以下两个条件就为true了。
1、任何具有有效名称的自主定制元素(Autonomous Custom Element)。
2、元素名是这些中的一个：article、aside、blockquote、body、div、footer、h1、
h2、h3、h4、h5、h6、header、main、nav、p、section、span。
Parameters:

extends4ClassName Class，元素类
extends4ElemName String，元素名
